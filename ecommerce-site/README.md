# Cali Clear Ecommerce Site

This is a static ecommerce site structure with reusable components, product data, and organized assets.

## Structure
- HTML pages for shop, product, cart, checkout, login, register, wishlist, about, contact, FAQ, terms, privacy, and order success.
- `assets/` for CSS, JS, images, and fonts.
- `data/` for static product data.
- `components/` for reusable HTML components.
- `pages/` for category and order-success pages.

## Usage
- Add your product data to `data/products.json`.
- Customize components in the `components/` folder.
- Update styles in `assets/css/` and scripts in `assets/js/`.
